## Usage

**Selection controls**

<ul>
  <li>Allow users to select options, make decisions and set preferences.</li>
  <li>Are to be visible and understandable at a quick glance.</li>
  <li>Are always accompanied with a clear label.</li>
</ul>

**Checkboxes**

<ul>
  <li>Make it easy to compare available options.</li>
  <li>
    Have been in user interfaces for a long time and should be used as expected.
  </li>
</ul>

```tsx
import { Checkbox } from '@equinor/amplify-components';

<Checkbox label="Label text" />;
```
